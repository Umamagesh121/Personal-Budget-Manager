# 💰 Personal Budget Manager

A modern and responsive **Personal Budget Manager** web application built with **HTML, CSS, and JavaScript**. This application helps users manage their personal finances by tracking income and expenses, calculating the current balance, and securely storing transaction data using the browser's Local Storage.

---

## 📖 Overview

Personal Budget Manager is a lightweight and user-friendly web application designed to simplify personal financial management. Users can add income and expense transactions, monitor their financial balance, and manage transaction history with a clean and responsive interface.

All transaction data is stored locally in the browser, ensuring that information remains available even after refreshing or reopening the application.

## 📋 How to Use

1. Enter a transaction description.
2. Enter the transaction amount.
   - Positive value (`1000`) → Income
   - Negative value (`-500`) → Expense
3. Click **Add Transaction**.
4. View the updated balance, income, and expenses.
5. Delete transactions when needed.
